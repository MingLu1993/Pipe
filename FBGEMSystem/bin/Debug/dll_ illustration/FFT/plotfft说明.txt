命名空间：plotfft
类名：Cplotfft
方法：[f, fft_data] = plotfft (x, fsample)
% INPUT PARAMETERS---------------------------------------------------------
% x:            输入信号(一维数组)
% fsample:      采样率（数值）
% OUTPUT VARIABLES---------------------------------------------------------
% f:            频率值(一维数组)
% fft_data:     幅值 (一维数组)

功能：求信号频谱。
